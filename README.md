## 一、網站簡介

本網站是一個為香港中學中文科設計的 **闖關式練習平台**，

現有題庫： 
1. 中三啟思出版社教科書題庫
2. 中三教圖出版社教科書題庫 （已出題，待更新） 
3. 中六範文闖關（已完成）

準備更新題庫： 
4. 中文答題知識（未完成）


學生可輸入角色名後進入不同等級關卡作答，系統會：

- 隨機抽出題目
- 自動計時（60 分鐘）
- 自動評分
- 顯示正確答案及解說
- 記錄歷史成績
- 解鎖下一關

同時設有 **教師後台**，方便老師查看：

- 學生成績
- 是否過關
- 作答時間
- 錯題內容
- 正確答案與解說

## 三、網站功能總覽

### 學生功能
- 輸入角色名開始闖關
- Lv1–100 解鎖式關卡
- 每關固定 30 題
- 每關建議 60 分鐘完成
- 倒數計時
- 時間到自動交卷
- 顯示對錯、正確答案及解說
- 可下載學生作答紙
- 可下載本關作答與答案

### 教師功能
- 教師後台密碼登入
- 查看成績總覽
- 按角色名搜尋
- 按過關／未過關篩選
- 查看學生錯題詳情
- 列印 / 另存為 PDF
